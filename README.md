# Jira Board Clone

Full Stack Jira-style board built with:

## Frontend
- React
- Tailwind CSS
- dnd-kit

## Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic

## Features
- Create Tasks
- Assignee & Assigner
- Due Dates
- Drag & Drop
- Status Persistence
- CRUD APIs
- Async FastAPI Endpoints

## Run Backend

```bash
uvicorn app.main:app --reload
